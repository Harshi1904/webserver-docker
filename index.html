🚀 Web Server using Docker

CodeAlpha Internship - Task 4

📌 About This Project

This project demonstrates how to deploy and manage a web server inside a Docker container. I have containerized a static website using Nginx and deployed it using Docker.

⚙ Technologies Used

  • Docker
  • Nginx Web Server
  • HTML & CSS
  • Command Line Interface

📖 What I Learned

  • Understanding Docker containerization basics
  • Building Docker images using Dockerfile
  • Running containers with port mapping
  • Monitoring container logs and lifecycle
  • Best practices in container-based deployment

🔄 Docker Commands Used

  • docker build - To build Docker image
  • docker run - To run container
  • docker ps - To list running containers
  • docker stop - To stop container
  • docker logs - To check logs

🌟 Future Improvements

In the future, this project can be enhanced by:

  • Deploying to cloud platforms like Azure or AWS
  • Using Docker Compose
  • Adding CI/CD pipeline integration
  • Creating dynamic applications using backend technologies
